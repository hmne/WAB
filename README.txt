‏‫WAB - WhatsApp Bot‬

‏‫WAB (WhatsApp Bot) هو روبوت متعدد الوظائف مصمم لتحسين تجربة المستخدم على تطبيق واتساب. يقدم البوت مجموعة من الميزات المفيدة والمبتكرة لتحسين التواصل والتفاعل بين المستخدمين.‬

‏‫WAB Logo‬

‫الميزات‬

‫يتضمن WAB العديد من الميزات المفيدة التي يمكنك استخدامها في محادثات واتساب الخاصة بك:‬

‫استخراج النص من الصور وملفات PDF: يمكن للبوت تحليل الصور وملفات PDF المرسلة واستخراج النص منها. لاستخدام هذه الميزة، أرسل الصورة أو الملف PDF مع كلمة "بوت حلل" أو قم بالرد على رسالة سابقة تحتوي على صورة أو ملف PDF بكتابة "بوت حلل".‬

‫تحديد اللغة: يمكن للبوت تحديد اللغة المستخدمة في النصوص المستخرجة من الصور وملفات PDF.‬

‫البحث عن رحلات جوية: يمكن للبوت البحث عن رحلات جوية باستخدام محرك بحث Skyscanner وتوفير معلومات حول الرحلات المتاحة والأسعار.‬

‫التعامل مع الأوامر: يمكن للبوت التعامل مع الأوامر المختلفة وتنفيذ الإجراءات المطلوبة بناءً على هذه الأوامر.‬

‫التفاعل مع API الخارجية: يدعم البوت التفاعل مع API الخارجية مثل OCR.Space API لاستخراج النص من الصور وتحويله إلى نص قابل للبحث.‬

‫استخدام تقنيات التعلم العميق: يمكن للبوت استخدام تقنيات التعلم العميق مثل OpenAI لتوفير ميزات متطورة مثل توليد النصوص وتحليل اللغة الطبيعية.‬

‫التفاعل مع مزودي الخدمات السحابية مثل AWS (أمازون ويب سيرفيس) لتوفير ميزات مثل التعرف على الكلام وتحويله إلى نص.‬

‫مساعدة البلد: يمكن للبوت استخدام ميزة CountryHelper للحصول على معلومات حول البلدان المختلفة وتقديم تفاصيل مثل العملة والعاصمة والسكان.‬

‫تحويل النص إلى كلام: يمكن للبوت استخدام مزودي الخدمات المختلفة لتحويل النص إلى كلام وقراءته بصوت عال.‬

‫التفاعل مع Whisper API: يمكن للبوت استخدام Whisper API للتعامل مع الميزات المتعلقة بالتعرف على الكلام وتحويله إلى نص.‬

‫استخدام التعلم الآلي المحلي: يمكن للبوت استخدام Whisper Local لتوفير ميزات التعرف على الكلام وتحويله إلى نص دون الاعتماد على خدمات سحابية.‬
